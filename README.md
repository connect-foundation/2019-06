## Prerequisites

- node 10.16.0
- npm 6.9.0

## Install & Usage

- Front-End

```sh
cd web
npm install
npm run build
npm start
```

- Back-End

```sh
cd server
npm install
npm start
```

## Run tests

```sh
npm run test
```
