## Prerequisites

- node 10.16.0
- npm 6.9.0

## Install & Usage
- Front-End
```sh
cd web
npm install
npm start
```
- Back-End
```sh
cd server
npm install
npm start
```
- Infra
```sh
cd infra
./init.sh
```

## Run tests
```sh
npm run test
```
